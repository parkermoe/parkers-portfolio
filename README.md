# parkermoesta.ai

Personal portfolio of Parker Moesta — Senior Machine Learning Scientist.

Static single-page site served via GitHub Pages. No frameworks, no build step:

- `index.html` — content, project dialogs, structured data
- `assets/css/main.css` — design tokens; dark evergreen (default) and alpine light themes
- `assets/css/fonts.css` + `assets/fonts/` — self-hosted Fraunces & Inter (variable woff2)
- `assets/js/main.js` — theme toggle, hero milestone rotator, scroll reveals, dialogs, filters

To preview locally: `python3 -m http.server` and open http://localhost:8000.
